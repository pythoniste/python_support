# 00_Concepts — Fondamentaux de la programmation système

Ce dossier ne contient **aucun programme système avancé**. C'est le
vocabulaire et les bases dont les dossiers suivants se servent ensuite
sans les redéfinir : qu'est-ce qu'un programme Python, comment on le
lance, comment il dialogue avec le système d'exploitation via ses
flux standards.

## Comment lire ce dossier

Lire les fichiers dans l'ordre numérique. Pour chaque module :

1. Lire le `.md` (1 à 2 pages).
2. Exécuter le `.py` correspondant et observer la sortie.
3. Passer au module suivant.

## Pré-requis

- Python 3.10 ou plus récent.
- Python de base : variables, conditions, boucles, fonctions, exceptions.
- Un terminal (Bash, Zsh, ou équivalent).
- **Aucune** connaissance système préalable n'est supposée.

## Validation et mise en pratique

Deux fichiers complètent les trois modules :

- **`QUIZ.md`** : cinq questions courtes pour auto-évaluer la
  compréhension, à utiliser après chaque module ou en fin de parcours.
- **`EXERCICES.md`** : quatre ateliers pratiques de synthèse, à faire
  **après** avoir étudié l'intégralité du dossier. Aucune installation
  tierce n'est requise.

Les corrigés sont disponibles dans le dossier `CORRIGES/`, à consulter
**après** avoir tenté les ateliers.

À l'issue de ces deux étapes, on peut ouvrir les chapitres suivants
et comprendre **chaque ligne** sans surprise.
